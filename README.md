# Entrega-1
Este repositorio es para realizar la entrega 1 donde se realizara un EDA que es un enfoque para analizar conjuntos de datos con el objetivo de resumir sus características y extraer información útil antes de realizar un modelado más profundo.

Nuestro EDA contendra: 

1. Carga de datos
2. Limpieza de datos
3. Analisis descriptivo
4. Visualizaciones graficas
5. Correlaciones
6. Insights
